## FAQ SECTION

**Badge:** FAQ

**Titulo:** Preguntas frecuentes

### Necesito Cianbox ERP para usar el POS?
Si, Cianbox POS esta disenado para integrarse con Cianbox ERP. La sincronizacion automatica de productos, precios y stock entre sistemas es la principal ventaja para cadenas que ya usan el ERP.

### Como funciona la sincronizacion entre sucursales?
Todas las sucursales se conectan a un panel central en la nube. Cuando cambias un precio, promocion o producto, se actualiza instantaneamente en todos los puntos de venta. Las ventas se sincronizan en tiempo real para que veas reportes consolidados al momento.

### Que pasa si una sucursal pierde internet?
El POS tiene modo offline. Si cae la conexion, la sucursal puede seguir vendiendo normalmente. Cuando vuelve internet, todas las ventas se sincronizan automaticamente con el sistema central sin perder ninguna transaccion.

### Puedo tener precios diferentes por sucursal o zona?
Si. Puedes crear multiples listas de precios y asignarlas a diferentes sucursales o zonas geograficas. Esto te permite tener precios diferenciados segun la ubicacion mientras mantienes el control centralizado.

### Cuanto tiempo toma implementar en toda la cadena?
Para cadenas de hasta 10 sucursales, la implementacion completa toma aproximadamente 2 semanas. Incluye configuracion, migracion de datos, capacitacion del personal y periodo de acompanamiento. Para cadenas mas grandes, trabajamos un plan de rollout por etapas.

### Que hardware necesito en cada sucursal?
Solo necesitas una PC o notebook con navegador web moderno y conexion a internet. Opcionalmente puedes agregar lector de codigo de barras USB, impresora de tickets termica y terminal MercadoPago Point. No requiere software especial ni servidores locales.
