## FAQ

### Preguntas y Respuestas

#### Cuanto tiempo toma la implementacion?
```
Para un supermercado de hasta 10 cajas, la implementacion completa toma entre 3-5 dias habiles. Esto incluye la sincronizacion de productos, configuracion de cajas y capacitacion del personal. Para cadenas mas grandes, el tiempo varia segun la cantidad de sucursales.
```

#### Funciona si se corta internet?
```
Si. Cada caja tiene modo offline. Si falla la conexion, la caja sigue funcionando con el catalogo completo de productos. Cuando vuelve internet, todas las ventas se sincronizan automaticamente con el servidor central.
```

#### Es compatible con mi lector de codigo de barras?
```
Si. Cianbox POS es compatible con cualquier lector USB o Bluetooth estandar. Tambien soportamos lectores seriales RS232 con adaptadores. Si tenes dudas sobre un modelo especifico, contactanos y lo verificamos.
```

#### Puedo integrar balanzas para productos por peso?
```
Si, en el plan Enterprise. Soportamos integracion con balanzas comerciales de las principales marcas (Systel, Kretz, Moretti, entre otras). Las balanzas pueden imprimir etiquetas con codigo de barras que el POS lee directamente.
```

#### Como se actualizan las ofertas semanales?
```
Desde el panel de administracion podes crear promociones con fecha de inicio y fin. Cuando llega la fecha, se activan solas en todas las cajas. Soportamos 2x1, segunda unidad al 50%, descuentos por categoria, combos, y ofertas tipo Black Friday o CyberMonday.
```

#### Que medios de pago soportan?
```
Efectivo, tarjetas de debito y credito (con integracion a terminales de pago), Mercado Pago (QR y Point), transferencias bancarias, y cualquier otro medio que quieras configurar. Tambien soportamos pagos mixtos.
```

#### Emiten factura electronica?
```
Si. Tenemos integracion nativa con AFIP para emision de facturas A, B y C. Tambien soportamos notas de credito y debito. El CAE se obtiene automaticamente al cerrar cada venta.
```

#### Que pasa si necesito mas cajas despues?
```
Podes agregar cajas en cualquier momento. Si superas el limite de tu plan, simplemente upgrade al siguiente. El cambio es inmediato y se prorratea el mes en curso.
```
