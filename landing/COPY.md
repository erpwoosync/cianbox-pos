## FAQ SECTION

**Badge:** FAQ

**Titulo:** Preguntas frecuentes

### Necesito tener Cianbox ERP para usar el POS?
Si, Cianbox POS esta disenado especificamente para integrarse con Cianbox ERP. La sincronizacion automatica de productos, precios y stock es la principal ventaja del sistema.

### Que hardware necesito?
Solo necesitas una computadora con navegador web moderno y conexion a internet. Opcionalmente puedes agregar lector de codigo de barras, impresora de tickets y terminal MercadoPago Point.

### Como funciona la sincronizacion con Cianbox?
Conectas tu cuenta Cianbox una vez y el sistema sincroniza automaticamente productos, categorias, marcas y listas de precios. Puedes sincronizar manualmente cuando quieras o programar sincronizaciones automaticas.

### Puedo usar MercadoPago para cobrar?
Si, soportamos MercadoPago Point para cobros con tarjeta y QR para pagos desde la app de MercadoPago. La integracion esta incluida en los planes Pro y Enterprise.

### Mis datos estan seguros?
Absolutamente. Cada cliente tiene sus datos completamente aislados. Usamos encriptacion en transito y en reposo, backups automaticos y servidores seguros. Cumplimos con las mejores practicas de seguridad.

### Tienen periodo de prueba?
Si, ofrecemos 14 dias de prueba gratuita con todas las funcionalidades. No necesitas tarjeta de credito para empezar. Te ayudamos con la configuracion inicial.
