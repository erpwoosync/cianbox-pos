Configuracion Simple

Conecta tu Cianbox ERP en 3 pasos

1

Obtene tu API Key

Desde tu panel de Cianbox ERP, genera una API Key para el punto de venta.

2

Configura la Conexion

Ingresa la URL de tu Cianbox y la API Key en el backoffice de Cianbox POS.

3

Sincroniza

Ejecuta la primera sincronizacion y tus productos estaran listos para vender.
