Facturacion Electronica AFIP

Emiti facturas A, B y C directamente desde el punto de venta. Configuracion simple, emision automatica y consulta de comprobantes.

Configurar facturacion AFIP
Comprobantes

Tipos de comprobantes soportados

Emiti el comprobante correcto segun el cliente.

A

Factura A

Para clientes Responsables Inscriptos. Discrimina IVA.

B

Factura B

Para consumidores finales y monotributistas. IVA incluido.

C

Factura C

Para monotributistas. Sin discriminar IVA.

Proceso

Emision desde el POS

1

Completa la venta

Agrega productos y selecciona el medio de pago como siempre.

2

Datos del cliente

Ingresa CUIT/DNI del cliente. El sistema determina el tipo de factura.

3

Emision automatica

El comprobante se genera y envia a AFIP. Recibis el CAE al instante.

4

Impresion o envio

Imprime el comprobante o envialo por email al cliente.

Beneficios

Por que facturar desde el POS

Ahorra tiempo

Emiti la factura en el mismo momento de la venta. Sin sistemas separados.

Menos errores

Los datos de la venta pasan directo a la factura. Sin recargar informacion.

Cumplimiento fiscal

Todos los comprobantes quedan registrados en AFIP automaticamente.

Historial completo

Consulta todos los comprobantes emitidos cuando los necesites.
